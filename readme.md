#How to start the Application
#open browser http://localhost:8080

start page --> Friends manager, click on View Friends list

http://localhost:8080/friend/list --> you will see the list of friends,
Then click on Add Friend, fill the form and click on Save button, you will see the new friend in the list.
You can also click the Delete button to delete a friend from the list.

http://localhost:8080/friend/add --> you will see the form to add a new friend, fill the form and click 
on Save button, you will see the new friend in the list.
